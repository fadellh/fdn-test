# FDN TEST

## Installation

1. Install package Manualy

```sh
 npm install
 npm install pg --save
```

2. Setting env Manually

```sh
export PORT=5000
export PG_HOST=localhost
export PG_PASS=postgres
export PG_USER=postgres
export PG_DB=fdn
export PG_PORT=5432
```

## Run Project Manually

```sh
npm run start
```

## Run Project with Docker

```sh
docker-compose up
```

## Testing Project

```sh
npm run test
```
